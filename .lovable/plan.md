

## 加密货币回收门店 H5 会员系统 — 开发计划

### 整体架构
基于 React + Tailwind CSS 构建 H5 移动端应用，使用 Lovable Cloud (Supabase) 作为后端，提供数据库、Edge Functions 和认证服务。

---

### 页面结构

#### 1. 首页 — "钩子页面"
- **数字会员卡展示**：深色拉丝金属质感卡片，动态显示溢价比例（如 `Premium +2.5%`）
- **实时汇率看板**：对比展示「市场价」vs「会员专属价」，高亮差价优势（BTC、ETH、USDT、USDC、BNB 等）
- **会员权益摘要**：用图标+文案快速传达核心价值——高于市价回收、即时结算、专属服务
- **醒目 CTA 按钮**：电光绿「立即注册会员」按钮

#### 2. 交易所选择页
- **大尺寸 Grid 宫格**：展示 Binance、OKX、Bybit、Bitget、Kraken 等交易所高清 SVG Logo
- 点击后进入该交易所的 API Key 填写流程

#### 3. API Key 填写与引导页
- **图文教程**：针对每个交易所展示「如何创建只读 API Key」的分步引导
- **安全提示**：醒目标注「只读权限 (Read-Only)」，说明仅用于账户验证，无法动用资产
- **输入框**：API Key + Secret Key 输入，下方实时显示格式校验状态
- **提交验证**：点击后调用 Edge Function 真实请求交易所接口验证有效性，按钮显示 loading → 成功绿色动画 / 失败红字提示（无弹窗）

#### 4. 会员面板（注册成功后）
- **专属会员卡**：显示会员卡号、所绑定交易所、注册时间
- **今日收购汇率表**：展示所有币种的会员专属收购价，tabular-nums 数字排版

#### 5. 店员汇率管理页（简单后台）
- 密码保护的简单管理页面
- 可手动修改各币种的今日收购价格
- 可新增/删除展示的币种

---

### 后端设计（Lovable Cloud / Supabase）

- **数据库表**：
  - `members` — 会员信息（交易所、API Key 加密存储、会员卡号、状态）
  - `exchange_rates` — 各币种收购汇率（币种、市场价、会员价、更新时间）
  - `exchanges` — 支持的交易所列表
- **Edge Functions**：
  - `verify-api-key` — 调用各交易所 API 验证 Key 有效性
  - `get-rates` — 获取当前汇率数据
  - `update-rates` — 店员更新汇率（需密码验证）

---

### 视觉风格
- 深碳黑背景 + 磨砂玻璃面板 + 金属拉丝纹理
- 电光绿 (#00FFC2) 仅用于核心 CTA 和价格高亮
- JetBrains Mono 数字字体 + Geist Sans 标题字体
- 所有动画 ≤ 300ms，丝滑但不拖沓
- 移动端优先，适配各种 H5 屏幕尺寸
